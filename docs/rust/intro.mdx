# Giriş

Rust, 2006 yılında temelleri atılan, 2009 yılında Mozilla tarafından desteklenmeye başlanan, 2015 yılında ilk stabil sürümü yayınlanan ve
söz dizimi C ve C++ dillerine benzeyen genel amaçlı bir programlama dilidir. Rust hız ve güvenliğe önem verir, bellek güvenliği ile ilgili
olası hataları derleme zamanında önleyerek uygulamanın ürün aşamasında kusursuz çalışabilmesini amaçlar.
Bunun gibi üstün özelliklerinden dolayı [Stack Overflow](https://insights.stackoverflow.com/survey/) web sitesinin anketlerinde üst üste 7 yıl "en sevilen programlama dili" olarak seçilmiştir.

Rust programlama dili günümüzde birçok alanda yaygın olarak kullanılmaktadır. Özellikle sistem programlama başta olmak üzere, komut satırı (CLI) araçları,
web uygulamaları, blok zinciri, veri sistemleri, gömülü sistemler, makine öğrenmesi, dağıtık sistemler ve oyun geliştirme gibi çok çeşitli alanlarda
kullanımı yaygınlaşmaktadır. Linux, çekirdeğinde C ve Assembly' den sonra üçüncü dil olarak Rust kullanmaya başlamıştır.

Rust programlama dilini kullanan bazı şirketler:

- Google
- Amazon
- Microsoft
- Meta
- Mozilla
- Npm
- Dropbox
- Cloudflare

## Blokzincir Ekosisteminde Rust

Blokzincir projelerinde güvenlik ve performans çok önemlidir. Rust programlama dili, sunduğu üst düzey güvenlik özellikleri, düşük enerji tüketimi
ve hızı sayesinde blokzincir projelerinde kullanımı gün geçtikçe artmaktadır. Gerek akıllı kontrat yazarken ,gerekse blokzincirin
çekirdek kodlarında Rust kullanan blokzincir projelerinden bazıları alfabetik sırayla şunlardır:

- Aptos
- Casper
- Cosmos
- Fuel Network
- Elrond
- Holochain
- Gear
- Internet Computer
- Near
- Polkadot
- Solana
- Sui Network
- Zksync
